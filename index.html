PokeWiki ReactJS

En este proyecto, he desarrollado una aplicación web llamada "PokeWiki" utilizando la biblioteca de JavaScript ReactJS. La aplicación está diseñada para permitir a los usuarios buscar, visualizar y explorar información sobre los Pokémon, como sus habilidades, estadísticas, tipos y evoluciones. Además, la aplicación cuenta con un sistema de filtrado y ordenamiento que facilita la búsqueda de los Pokémon según las preferencias del usuario.
